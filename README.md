# Packwise

Expo + React Native mobile app for Packwise.

## Run

```bash
npm install
npm run dev
```

Then scan the QR code with Expo Go, or run:

```bash
npm run ios
npm run android
npm run web
```

## Environment

Create `.env.local` with:

```env
EXPO_PUBLIC_SUPABASE_URL=your-supabase-url
EXPO_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
```

The app currently contains the native login screen only.
